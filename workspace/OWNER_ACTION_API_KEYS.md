# オーナーアクション依頼: API Key取得

依頼日: 2026-02-01
期限: 2026-02-07

## 概要
マーケティング自動化のため、以下のAPIキーが必要です。

## 1. GitHub Token
URL: https://github.com/settings/tokens
Scope: repo, workflow, read:org
所要時間: 5分

## 2. Discord Bot Token
URL: https://discord.com/developers/applications
所要時間: 10分

## 3. Slack Webhook
URL: https://api.slack.com/apps
所要時間: 10分

詳細手順: docs/API_KEY_COMPLETE_GUIDE.md
