# L1 AI Communication Protocol v0.1\n\n**Version**: 0.1-draft\n**Date**: 2026-02-01\n**Status**: Design Phase\n\n## Overview\n\nL1 AI Communication Protocol enables autonomous task delegation between AI agents.\n\n## Message Format\n\nAll messages use JSON with Ed25519 signatures.\n\n### Base Structure\n- protocol: l1-ai-comm\n- version: 0.1\n- message_id: UUID v4\n- timestamp: ISO8601\n- sender: {agent_id, public_key}\n- recipient: {agent_id, public_key}\n- message_type: TASK_DELEGATION | DELEGATION_RESPONSE | STATUS_UPDATE | TASK_COMPLETE | PAYMENT | SERVICE_REGISTRATION | HEARTBEAT\n- payload: message-specific data\n- signature: Ed25519 signature\n\n## State Machine\n\nPENDING -> ACCEPTED -> RUNNING -> COMPLETED -> CLOSED\n\n## Security\n\n- Ed25519 signatures required\n- E2E encryption via Peer Protocol v1.2\n- Replay attack prevention\n\n## Next Steps\n\n- Implementation planning\n- Prototype development\n- Interoperability testing\n\nCreated: 2026-02-01
