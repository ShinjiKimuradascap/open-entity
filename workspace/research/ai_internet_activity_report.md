## 2. AIがネットで活動する方法

### 2.1 自律的ウェブブラウジング

**主要ツール**:
| ツール | 特徴 | 用途 |
|--------|------|------|
| Selenium | 多言語対応、歴史が長い、W3C準拠 | レガシーシステムのテスト |
| Playwright | Microsoft製、高速、モダンなWebアプリ向け | React/Vue/Angular対応 |
| Puppeteer | Chrome DevTools Protocol使用 | ヘッドレスブラウジング |

**AIによる自動化の進化**:
- セルフヒーリング: レイアウト変更に自動対応
- 視覚的テスト: UI変更のAI検出
- 自然言語生成: 「〜を検索して」という指示でスクリプト生成

### 2.2 SNS自動化

**プラットフォーム別API状況 (2024年)**:

| プラットフォーム | API状況 | コスト | 制限 |
|------------------|---------|--------|------|
| Reddit | PRAW (Python) | 有料化 (2023~) | アンチスパム厳格 |
| Twitter/X | API v2 | $149~/月 (Basic) | 無料枠大幅縮小 |
| Discord | Bot API | 基本無料 | 比較的寛容 |

**主要ライブラリ**:
- Reddit: praw
- Twitter: tweepy
- Discord: discord.py / discord.js

### 2.3 AIエージェントの活動パターン

**現在の能力**:
1. 情報収集: 自動検索、要約、データ抽出
2. コンテンツ生成: 文章、画像、動画の自動作成
3. エンゲージメント: 返信、いいね、フォローの自動化
4. モデレーション: スパム検出、不適切コンテンツの除去

**将来の予測 (2025年)**:
- より高度な自律性（複雑なタスクの自動実行）
- リアルタイムパーソナライゼーション
- マルチモーダル対応（テキスト+画像+音声）
- 規制の強化（AI生成コンテンツのラベリング義務化）
