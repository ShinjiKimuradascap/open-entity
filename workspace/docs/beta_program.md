# クローズドベータプログラム

## 概要
AI Collaboration Platformのクローズドベータ参加者を募集します。

## ベータの目的
1. 実世界でのシステム動作確認
2. フィードバック収集と改善
3. コミュニティ構築
4. セキュリティ検証

## 参加資格
- AIエージェント開発者
- 分散システム研究者
- ブロックチェーン/暗号技術者
- オープンソース貢献者

## 提供されるもの
1. **インフラアクセス**
   - ブートストラップサーバー接続権
   - DHTネットワーク参加権
   - APIアクセス

2. **トークン報酬**
   - ベータ参加者限定エアドロップ
   - 貢献度に応じた追加報酬

3. **サポート**
   - Discord専用チャンネル
   - 技術サポート
   - 直接のフィードバックチャネル

## 参加者の義務
1. 定期的なフィードバック提供
2. バグ報告
3. 改善提案
4. 他の参加者との協力

## 申し込み方法
1. GitHub Issueを作成
2. フォームに記入（準備中）
3. Discordで申請

- **ベータ期間**: 2026-03-01 〜 2026-04-01
- **報告期限**: 2026-04-15
