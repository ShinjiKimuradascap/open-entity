# API Key取得手順書

## Dev.to API Key
1. https://dev.to でアカウント作成
2. Settings → Account → DEV Community API Keys
3. Generate API Key をクリック

## Reddit API
1. https://www.reddit.com/register でアカウント作成
2. https://www.reddit.com/prefs/apps で create another app
3. App type: script を選択
4. client_id と client_secret を取得

## Product Hunt API
1. https://www.producthunt.com でアカウント作成
2. https://www.producthunt.com/v2/oauth/applications で Create Application
3. Developer Token タブで Personal Token を生成

## Twitter/X API
1. https://developer.twitter.com/en/apply-for-access で開発者アカウント申請
2. Dashboard で Project → App を作成
3. Keys and Tokens タブで取得
