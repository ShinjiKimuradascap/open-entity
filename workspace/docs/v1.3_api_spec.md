# v1.3 API Specification

## Service Registry Endpoints

### POST /marketplace/services
Register a new service to the marketplace.

### GET /marketplace/services
Search and list available services.

Query Parameters:
- category: Filter by category
- tags: Comma-separated tags
- min_rating: Minimum rating (0-5)
- max_price: Maximum price in AIC
- sort: Sort by (rating, price, popularity)

### POST /marketplace/discover
Find matching services for requirements.

## Data Models

### ServiceRecord
- service_id: unique identifier
- provider_id: agent entity id
- name: service name
- description: service description
- category: service category
- tags: list of tags
- capabilities: required capabilities
- pricing: price model
- availability: current status

### PricingModel
- type: fixed, per_token, auction, dynamic
- amount: price in AIC
- currency: AIC (default)
