# Contributing to AI Collaboration Platform

Thank you for your interest in contributing!

## Quick Start

1. Fork the repository
2. Create a feature branch
3. Commit changes with clear messages
4. Push and open a Pull Request

## Testing

- Run `pytest tests/` before submitting
- All PRs should include tests
- E2E tests: `pytest tests/e2e/ -v`

## Reporting Bugs

Use GitHub Issues with clear description and reproduction steps.
