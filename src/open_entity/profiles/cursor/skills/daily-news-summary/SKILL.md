# Daily News Summary Skill

「今日の主要なニュース」を検索・抽出し、簡潔で分かりやすいサマリーを作成するためのスキルです。

## 概要
複数の信頼できるニュースソースから最新の情報を収集し、政治、経済、テクノロジー、エンタメなどのカテゴリ別に整理して報告します。

## ガイドライン
1. **情報の鮮度**: 常に当日の最新ニュースを優先します。
2. **信頼性**: 公的な通信社や大手新聞社の情報を優先し、出典（URL）を明記します。
3. **客観性**: 主観的な意見を排除し、事実に基づいた要約を行います。
4. **経済・社会情報の高度化 (国別サマリー)**:
    経済カテゴリの調査では、単なる要約ではなく以下の **[国別レイヤー構造]** でサマリーを作成します。
    対象国（JP, US, UK, IN）ごとに以下の3点を必須とします：

    - **① 国債利回りの網羅（超長期）**:
        **10年、20年、30年** の利回りを必ず調査し、前日比やイールドカーブの変化を記載する。
    - **② 主要銘柄の動向**:
        指数だけでなく、その国を代表する主要銘柄（例: トヨタ、三菱UFJ、NVIDIA、テスラ、HSBC、アストラゼネカ等）の個別ニュースを必ず含める。
    - **③ 国別の地政学・社会情勢**:
        金融政策だけでなく、その国の政治、法規制、地政学的リスク、社会トレンドを経済と関連付けて報告する。

5. **構造化**: 
    「マーケット概況」で全体の数字を示した後、セクションを **「国別（Japan, USA, UK, India）」** に分け、各国の「金利・銘柄・情勢」を統合してまとめます。
