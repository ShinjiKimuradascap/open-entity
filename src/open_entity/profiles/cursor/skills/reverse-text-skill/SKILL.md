# Reverse Text Skill

This skill provides a tool to reverse text strings.

## Usage

Use this skill when users need to:
- Reverse a text string
- Reverse a sentence or paragraph
- Reverse any textual content

## Tool

The skill provides the `reverse_string` tool which takes a text string and returns it reversed.

## Example

To reverse "Hello, World!", call the `reverse_string` tool with the argument `{"text": "Hello, World!"}` which will return `"!dlroW ,olleH"`.
