# Microsoft AutoGen Adapter

Open Entity A2A Protocol integration with Microsoft AutoGen.

## Overview

This adapter enables AutoGen agents to connect to Open Entity's A2A decentralized network.

## Requirements

pip install pyautogen open-entity

## Quick Start

Basic usage example for connecting AutoGen agents to A2A network.

## API Reference

### AutoGenConversableAgentWrapper

Extends AutoGen ConversableAgent with A2A capabilities.

### A2AConfig

Configuration options for A2A integration.

### A2ATool

Tool wrapper for AutoGen function calling.

## Architecture

AutoGen Agent connects to AgentRegistry which connects to Bootstrap Node.
